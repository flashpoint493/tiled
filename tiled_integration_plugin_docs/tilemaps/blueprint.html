Create a Blueprint

1.- The first step is to import your Tile Map asset, to do that pleasae follow the guide in Import a Tile Map.

2.- Once you have the Tile Map asset ready, we can create a blueprint instance for it using one of the following methods

  • Right Click on the Asset > Tiled Integration > Create Blueprint.

  • Create a actor blueprint with the ATITileMapActor class or a children class from it, then open the blueprint and select the Render Component and set the Tile Map property to use the asset you want and finally compile and save.

Note: Changing the Tile Map property from the blueprint won’t update the preview window, to force an update, close and reopen the blueprint editor.

Note: Do not add a Tile Map Component manually to any blueprints as it is not supported and will have unexpected behavior.
